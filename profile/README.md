- Build automated systems for the Chalupa

  - The Chalupa should be able to operate in space
- Build a community around the Chalupa
  - The Chalupa should be welcoming to all
  - The Chalupa should be helpful to all
  - The Chalupa should be respectful to all
  - The Chalupa should be inclusive to all
